# 🔢 Нумерологія · Магія Чисел

Сайт про нумерологію українською мовою. Одна HTML-сторінка без залежностей, готова до розміщення на GitHub Pages.

## ✨ Функціонал

- **Калькулятор числа долі** — введіть дату народження, отримайте число та його опис
- **Калькулятор числа імені** — введіть ім'я українською, отримайте «число виразу»
- **Таблиця чисел 1–9** з архетипами та ключовими словами
- **Анімований зоряний фон** (Canvas API)
- **Пояснення методології** розрахунку
- Чітке **застереження**, що нумерологія — не наука

## 🌐 Розміщення на GitHub Pages

1. Завантажте репозиторій на GitHub
2. Відкрийте **Settings → Pages**
3. У розділі **Source** оберіть гілку `main`, папку `/ (root)`
4. Натисніть **Save** — сайт буде доступний за адресою `https://ваш-нікнейм.github.io/назва-репозиторію`

Уся інформація на сайті призначена **виключно для розваги та ознайомлення**.  
Нумерологія — це **езотерична практика, а не наука**. Не слід приймати важливих рішень на підставі нумерологічних розрахунків.
